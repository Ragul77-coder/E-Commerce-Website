## 🤖 How I used AI
I designed the HTML page structures and compiled the custom styling system in `index.css`. The Antigravity AI assistant helped me write the centralized cart logic in `app.js` utilizing `localStorage` for persistence, configure the Clerk dynamic script loader for Google Sign-in, and organize the 15-item product database in `products.js`.

## 📚 What I learned
- Integrating third-party authentication services (Clerk JS SDK) in a pure static Vanilla HTML/JS frontend application.
- Structuring fluid grid layouts and pillow-soft components using CSS custom properties for cohesive brand identity.
- Managing cart state changes (addition, quantity changes, subtotal calculations) synchronously in client-side storage.
